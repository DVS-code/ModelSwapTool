# Tools Folder

This folder is used by FH5 ModelSwap Tool at runtime.

## Required
- `quickbms.exe` (auto-downloaded on first run)
- `zip.bms` (bundled placeholder, replace with a working script if needed)

## Bundled placeholders
- `zip.bms`
- `grub_parser.bt`

## Optional external tools
These are not bundled or installed automatically:
- 010 Editor: https://www.sweetscape.com/010editor/
- 3DSimED: https://www.sim-garage.co.uk/3dsimed/

## Notes
- QuickBMS will be downloaded automatically when missing.
- Replace `zip.bms` with a correct script compatible with FH5 archives.
